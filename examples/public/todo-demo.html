How to Test Incremental Rendering

This demo shows O(delta) rendering - only changed DOM nodes are touched:

  • Add items - Watch only new elements being created
  • Remove middle - See single REMOVE operation, not re-render
  • Toggle done - Click checkbox to see attribute updates only
  • Cycle priority - Click priority badge to change (low/medium/high)
  • Filter - Uses ifilter combinator for reactive filtering
