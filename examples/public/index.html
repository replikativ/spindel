Spindel Examples

Delta-direct incremental DOM rendering without virtual DOM diffing

What is Spindel?

Spindel is an incremental UI rendering library for ClojureScript that achieves true O(delta) DOM updates. Unlike React's virtual DOM diffing, spindel tracks changes at the data level and produces precise DOM mutations.

Key concepts: signal (reactive state), track (observe changes), spin (reactive computation), ifor-each (incremental list rendering).

Infinite Scroll

Virtual list with 10,000 items but only ~10 rendered. Scroll produces minimal DOM changes.

Versioned Editor

Getting Started

# Clone the repository
git clone https://github.com/replikativ/spindel.git
cd spindel/examples

# Install dependencies
npm install

# Start all examples with one command
npx shadow-cljs watch examples

# Open http://localhost:8080/
