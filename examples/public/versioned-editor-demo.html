Versioned editor — branch as inference experiment

Reactive UI that projects between trunk and forks. Click Try to spawn a speculative experiment; accept it to merge, reject to discard.

State is a single spindel signal map of {:branch-name [blocks]}. The active branch is another signal. The page derives a (diff main, active) per-block classification and renders gutters / strikethrough / dim accordingly — all reactive, all incremental. Inspect @examples.versioned-editor-demo/branches and @examples.versioned-editor-demo/active-branch in the CLJS REPL on port 9631.
