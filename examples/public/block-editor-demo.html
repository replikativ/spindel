Spindel Block Editor

Hierarchical block editor with O(delta) incremental rendering

Keyboard Shortcuts

  • Enter - Create a new block after the current one
  • Backspace (on empty block) - Delete the block
  • Tab - Indent block (make it a child of the previous sibling)
  • Shift+Tab - Outdent block (make it a sibling of its parent)
  • Arrow Up/Down - Navigate between blocks
  • Click +/- - Collapse/expand blocks with children
